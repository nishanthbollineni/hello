CSS font-style Property

Example

Set different font styles for three paragraphs:

p.a {
  font-style: normal;
}

p.b {
  font-style: italic;
}

p.c {
  font-style: oblique;
}
Try it Yourself »

Definition and Usage

The font-style property specifies the font style for a text.


CSS Syntax

font-style: normal|italic|oblique|initial|inherit;

Property Values
